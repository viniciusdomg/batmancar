# batmancar

O projeto “Carro do Batman” consiste no desenvolvimento de um sistema integrado envolvendo um carro robô controlado por um Arduino ESP8266 e um aplicativo móvel. A comunicação entre o aplicativo e o carro ocorre via Wi-Fi, utilizando o Firebase como plataforma intermediária para envio e leitura de comandos em tempo real.
